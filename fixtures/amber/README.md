# AMBER Fixtures

This directory contains small, deterministic fixtures used by the Rust tests.

## acetamide_tiny

`fixtures/amber/acetamide_tiny/*/acetamide.prod.out` are trimmed AMBER outputs
derived from `/gibbs/helmut/projects/flexible_ligands_GIST/acetamide/dG_solv/TI/flexible`.

Each window contains:
- full control/header sections needed by alchemlyb parsers
- 20 MBAR Energy analysis blocks (evenly sampled across the run)
- 20 DV/DL blocks with unique NSTEP values

These fixtures are used by:
- `crates/alchemrs-parse` AMBER integration tests
- `crates/alchemrs-prep` decorrelation comparisons against alchemlyb
- `crates/alchemrs-estimators` TI comparisons against alchemlyb

### Regenerating the fixtures

The tiny fixtures are generated by slicing the original AMBER output files.
Use the same extraction script pattern used in the test setup to:
1. read the source `acetamide.prod.out`
2. copy required header sections
3. include 20 MBAR blocks and 20 DV/DL blocks with unique NSTEP values

### Regenerating alchemlyb comparison outputs

Using the `alchemlyb` conda environment, run:

```
source $(conda info --base)/etc/profile.d/conda.sh
conda activate alchemlyb

python - <<'PY'
from alchemlyb.parsing.amber import extract_dHdl, extract_u_nk
from alchemlyb.preprocessing.subsampling import decorrelate_dhdl, decorrelate_u_nk
from alchemlyb.estimators import TI
import pandas as pd
from pathlib import Path

base=Path('fixtures/amber/acetamide_tiny')

# decorrelated dhdl for 0.1
df=extract_dHdl(base/'0.1'/'acetamide.prod.out', T=300.0)
decorrelate_dhdl(df).reset_index().to_csv(base/'0.1'/'dhdl.decorrelated.csv', index=False)

# decorrelated u_nk for 0.1
df=extract_u_nk(base/'0.1'/'acetamide.prod.out', T=300.0)
decorrelate_u_nk(df, method='dE').reset_index().to_csv(base/'0.1'/'u_nk.decorrelated.csv', index=False)

# TI delta for 0.0 -> 1.0 (all windows)
paths=sorted([p/'acetamide.prod.out' for p in base.iterdir() if p.is_dir()], key=lambda p: float(p.parent.name))
frames=[extract_dHdl(p, T=300.0) for p in paths]
dhdl=pd.concat(frames)
result=TI().fit(dhdl)
states=result.states_
idx0=states.index((0.0,)) if isinstance(states[0], tuple) else states.index(0.0)
idx1=states.index((1.0,)) if isinstance(states[0], tuple) else states.index(1.0)
value=result.delta_f_.iloc[idx0, idx1]
(base/'ti_0.0_1.0.delta_f.txt').write_text(f\"{value:.8f}\\n\")
PY
```
